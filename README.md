# Система мониторинга и анализа трудоустройства выпускников

Веб-приложение для автоматизации процессов трудоустройства выпускников вузов, сбора аналитики, генерации официальной отчетности и проведения предварительных собеседований с использованием Искусственного Интеллекта.

> **Магистерская диссертация** | Санкт-Петербургский государственный технологический институт (технический университет)

## Ключевые возможности

### Экосистема для ВУЗа
- **Центр отчетов:** Автоматическая генерация и скачивание документов в PDF (Форма ВПО-1, Мониторинг КЦСТ, Списки воинского учета) с возможностью живого предпросмотра (A4).
- **AI-Аналитик:** Нейросеть анализирует большие данные по трудоустройству, выявляет студентов в "зоне риска" и автоматически пишет текстовые заключения для отчетов.
- **Мониторинг:** Интерактивный дашборд с KPI, демографией, географией трудоустройства и рейтингом партнеров-работодателей.
- **Управление документами:** Загрузка официального логотипа вуза и печати для их автоматической подстановки в генерируемые документы.

### Интеграция с ИИ (Llama 3.3 via Groq)
- **AI Карьерный консультант:** Чат-бот, который анализирует профиль студента, рекомендует подходящие вакансии и помогает подготовиться к интервью.
- **Авто-собеседования:** Генерация уникальных технических вопросов под конкретную вакансию, автоматическая проверка ответов студента и выставление оценки.
- **Парсинг навыков:** Автоматическое извлечение тегов компетенций из описания вакансий.
- **Конструктор резюме:** Помощь в составлении резюме (красивое описание навыков, предыдущих мест работ и достижений).

### Продвинутый Профиль и UX
- **Drag-and-Drop интерфейс:** Полностью настраиваемая сетка профиля — пользователи могут перетаскивать виджеты (Roadmap, Карьера, Резюме, Статистика) по своему усмотрению.
- **Персонализация:** Сохранение расположения блоков профиля в LocalStorage.
- **Портфолио:** Расширенная анкета, таймлайн карьеры, история обучения и верификация данных.
- **Система наград:** Отслеживание пройденных Roadmap's и начисление баллов (XP) с последующими наградами для улучшения профиля.

### Коммуникация и Мессенджер
- **Единый чат:** Обмен сообщениями между студентами, работодателями и сотрудниками вуза.
- **Функции HR:** Назначение технических интервью (календарь) и отправка приглашений на видеозвонки прямо в чате.
- **Контекст:** Быстрый просмотр профиля собеседника (Modal) без перехода на другую страницу.

### Безопасность и Доступ
- **Ролевая модель:** Студент / Работодатель / Сотрудник ВУЗа / Администратор.
- **OAuth 2.0:** Вход через Google и GitHub.
- **Email Verification:** Подтверждение регистрации через SMTP (Gmail).
- **JWT Auth:** Безопасная сессия с защитой маршрутов.

## Технологический стек

**Frontend:**
- **Vue 3** (Composition API) + **Vite**
- **Ant Design Vue** (UI Framework)
- **VueDraggable** (Drag-and-Drop функционал)
- **Html2Pdf.js** (Генерация PDF отчетов)
- **Chart.js** (Визуализация данных)
- **Axios** (API Client)
- **Marked** (Markdown рендеринг)

**Backend:**
- **Node.js** + **Express**
- **PostgreSQL** (Сложная реляционная структура: `users`, `graduates`, `university_staff`, `employment_records` и др.)
- **Multer** (Загрузка файлов, аватаров, документов)
- **Nodemailer** (Email-рассылки)
- **OpenAI SDK** (Интеграция с LLM Groq)

## Установка и запуск

### Предварительные требования
- Node.js (v16+)
- PostgreSQL

### 1. Клонирование репозитория
```bash
git clone https://github.com/YOUR_USERNAME/it-graduate-employment-monitoring.git
cd it-graduate-employment-monitoring
```

### 2. Установка зависимостей
Скрипт автоматически установит библиотеки для корня, бэкенда и фронтенда:
```bash
npm run install-all
```

### 3. Настройка базы данных
- Создайте базу данных `graduate_monitoring` в PostgreSQL.
- Импортируйте структуру таблиц из папки `/database` (включая новые таблицы `university_staff`, `university_reports`).

### 4. Настройка переменных окружения (.env)
Создайте файл `backend/.env` и заполните его по образцу:
```env
PORT=4000
# База данных
DB_USER=postgres
DB_PASSWORD=ваш_пароль
DB_HOST=localhost
DB_PORT=5432
DB_NAME=graduate_monitoring

# Безопасность
JWT_SECRET=ваш_супер_секретный_ключ

# AI (Groq Cloud)
GROQ_API_KEY=gsk_...

# Почта (Gmail App Password)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=ваш_email@gmail.com
SMTP_PASS=ваш_пароль_приложения

# OAuth
GOOGLE_CLIENT_ID=...
GOOGLE_CLIENT_SECRET=...
GITHUB_CLIENT_ID=...
GITHUB_CLIENT_SECRET=...
```

### 5. Запуск проекта
Запуск Frontend и Backend одной командой из корня проекта:

```bash
npm run dev
```
- **Frontend:** http://localhost:3000
- **Backend:** http://localhost:4000
